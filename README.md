# picasso-test

Реализована загрузка постов с JSPlaceholder  
Посты подгружаются по мере прокрутки  
Обрезано описание  
Для просмотра поста есть кнопка

## Стек

-Redux
-RTK Query
-React-router v6

### Установка `npm install`

### Запуск `npm start`
